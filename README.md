# Student Grade Management System (C语言实现)

一个使用C语言实现的动态学生成绩管理系统，支持任意数量的学生和课程。这是我大学C语言课程实验项目，获得了95分的好成绩。

# ✨ 特性

- 🎯 **完全动态**: 运行时决定学生数量和课程数量
- 📊 **多维度统计**: 计算学生平均分、科目平均分、最高分
- 🏆 **智能分析**: 自动找出最优秀学生
- 🛡️ **内存安全**: 使用动态内存分配和释放
- 💾 **输入验证**: 基本的输入验证和错误处理
- 📋 **清晰输出**: 格式化的结果显示

## 使用办法个结果
1. 先输入一个班的学生数量，他们上了几门课
2. 输入每一个的消息，有名字，学号，每门课的成绩
3. 输出四个部分的成绩分析：
   - 每个学生的成绩和平均分
   - 每门课的平均分
   - 最高分是谁得了哪一门课
   - 最优秀的学生（平均分最高）
